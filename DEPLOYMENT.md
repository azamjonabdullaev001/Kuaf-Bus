# Краткое руководство по развертыванию

## Шаг 1: Установка зависимостей

### Windows:
```bash
start.bat
```

### Linux/Mac:
```bash
chmod +x start.sh
./start.sh
```

## Шаг 2: Доступ к системе

### Админ-панель
1. Откройте браузер: http://localhost:3000
2. Войдите с учетными данными:
   - University ID: `ADMIN001`
   - Password: `admin123`
3. Создайте студентов и водителей

### Мобильное приложение
1. Откройте терминал в директории `bus/`
2. Запустите: `npx expo start`
3. Отсканируйте QR-код через Expo Go приложение
4. Войдите с созданными учетными данными

## Шаг 3: Тестирование Real-time

1. Создайте несколько водителей через админ-панель
2. Войдите как водитель в мобильном приложении
3. Дайте разрешение на геолокацию
4. Откройте карту - ваша позиция будет обновляться каждую секунду
5. Войдите как студент на другом устройстве
6. Студент увидит всех водителей на карте в реальном времени

## Полезные команды Docker

### Просмотр запущенных контейнеров
```bash
docker-compose ps
```

### Просмотр логов
```bash
docker-compose logs -f
```

### Остановка всех сервисов
```bash
docker-compose down
```

### Перезапуск сервисов
```bash
docker-compose restart
```

### Очистка всех данных
```bash
docker-compose down -v
```

## Проблемы и решения

### Backend не запускается
Проверьте, что PostgreSQL запустился:
```bash
docker-compose logs postgres
```

### Админ-панель не подключается к API
Убедитесь, что backend работает:
```bash
curl http://localhost:8080/api/admin/students
```

### Мобильное приложение не подключается
1. Измените IP в `bus/config.js` на IP вашего компьютера
2. Перезапустите Expo: `npx expo start --clear`

## Настройка для production

1. Измените `JWT_SECRET` в `.env`
2. Установите SSL сертификаты
3. Настройте CORS для вашего домена
4. Измените пароли базы данных
5. Добавьте Google Maps API ключ для Android
