## 🎯 Как Это Работает

### Без Мерцания:
1. WebView создаётся **один раз** при монтировании компонента
2. HTML хранится в `htmlContentRef.current`
3. При изменении маркеров отправляется `postMessage` в WebView
4. JavaScript внутри WebView обновляет маркеры **без перезагрузки**
5. Leaflet.js плавно двигает существующие маркеры через `smoothMove`

### Маркеры:
1. При первом добавлении создаётся L.marker с иконкой
2. При обновлении позиции вызывается `smoothMove` (анимация 800ms)
3. Если расстояние < 1 метр - не обновляется (экономия ресурсов)
4. Старые маркеры автоматически удаляются

### Zoom:
1. При первой загрузке: `zoom: 16`, центрирование после 1 секунды
2. Пользователь может зумить/перемещать карту свободно
3. При обновлении координат zoom **НЕ меняется**
4. Кнопка 📍 возвращает к пользователю с **текущим zoom**
