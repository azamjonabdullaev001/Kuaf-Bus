# Настройка Яндекс Карт для KUAF BUSS

## Получение API ключа

Чтобы использовать Яндекс Карты в приложении, вам нужно получить API ключ:

1. Перейдите на сайт: https://developer.tech.yandex.ru/
2. Войдите в свой аккаунт Яндекс (или создайте новый)
3. Перейдите в раздел "JavaScript API и HTTP Геокодер"
4. Нажмите "Получить ключ"
5. Заполните форму и получите API ключ

## Установка API ключа

1. Откройте файл `bus/components/YandexMap.js`
2. Найдите строку:
   ```javascript
   <script src="https://api-maps.yandex.ru/2.1/?apikey=YOUR_API_KEY&lang=ru_RU"
   ```
3. Замените `YOUR_API_KEY` на ваш реальный API ключ

## Альтернатива для тестирования

Для тестирования без API ключа закомментируйте параметр `apikey`:
```javascript
<script src="https://api-maps.yandex.ru/2.1/?lang=ru_RU"
```

**Важно:** Без API ключа карты будут работать, но с ограничениями по количеству запросов.

## Использование

Приложение автоматически:
- Показывает Яндекс Карты на мобильных устройствах (iOS/Android)
- Отображает заглушку с информацией о водителях на веб-версии
- Использует реальное отслеживание через WebSocket
