## 🐛 ОТЛАДКА:

### В Expo Dev Tools (терминал):
- Нажмите `j` чтобы открыть Chrome DevTools
- Смотрите консоль для логов: "Attempting login for:", "Login response:" и т.д.

### Если ошибка "Network Error":
- Backend не запущен или недоступен
- Неправильный IP адрес
- Брандмауэр блокирует соединение

### Если ошибка "Превышено время ожидания":
- Backend слишком медленно отвечает
- Проблемы с сетью

### Если ошибка "Неверный ID":
- Пользователь не существует в базе
- Проверьте через админ-панель или создайте нового
