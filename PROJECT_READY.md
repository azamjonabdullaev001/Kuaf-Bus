## 📱 КАК СОЗДАТЬ APK

### Быстрая инструкция:

1. **Обновите URL в `bus/config.prod.js`**:
   ```javascript
   const PRODUCTION_API_URL = 'https://ваш-ngrok-url.ngrok-free.dev/api';
   const PRODUCTION_WS_URL = 'wss://ваш-ngrok-url.ngrok-free.dev/ws';
   ```

2. **Запустите build**:
   ```bash
   cd bus
   eas build -p android --profile preview
   ```

3. **Дождитесь завершения** (10-20 минут)

4. **Скачайте и установите APK** на телефон

📖 **Полная инструкция**: [APK_BUILD_GUIDE.md](APK_BUILD_GUIDE.md)

## 🚀 РАЗВЁРТЫВАНИЕ НА PRODUCTION

### Что нужно сделать:

1. **Подготовить сервер** (следуйте [PRODUCTION_GUIDE.md](PRODUCTION_GUIDE.md))

2. **Обновить пароли**:
   ```bash
   ./generate-secrets.sh
   ```

3. **Развернуть**:
   ```bash
   ./deploy-production.sh
   ```

4. **Создать пользователей с паролями**:
   ```bash
   docker exec -i bus-tracking-db-prod psql -U bususer -d bus_tracking < create-test-users-with-passwords.sql
   ```

5. **Обновить config.prod.js** с URL production сервера

6. **Создать APK**:
   ```bash
   cd bus
   eas build -p android --profile production
   ```
