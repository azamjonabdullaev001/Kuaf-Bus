# KUAF BUSS - Новые возможности

## ✅ Реализованные функции

### 1. 🌍 Экран выбора языка
- **Красивый градиентный дизайн** (красный → черный)
- **Логотип KUAF BUSS** с анимацией
- **Два языка**: Русский и O'zbekcha (Узбекский)  
- **Появляется при первом запуске** приложения
- Сохраняет выбор пользователя

### 2. 🔐 Автоматический вход
- **Автосохранение данных входа** после первого логина
- **Автоматический вход** при следующем запуске
- **Безопасное хранение** токена и данных пользователя
- При выходе язык остается выбранным

### 3. 🗺️ Яндекс Карты
- **Заменены Google Maps на Яндекс Карты**
- **WebView интеграция** через Yandex Maps API
- **Real-time отслеживание** водителей и студентов
- **Координаты Ташкента** по умолчанию (41.3111, 69.2797)
- **Анимированная камера** для плавного перемещения

### 4. 🌐 Полный перевод
Все экраны переведены на оба языка:
- ✅ Экран выбора языка
- ✅ Экран входа
- ✅ Карта (студенты)
- ✅ Карта (водители)
- ✅ Профиль
- ✅ Навигационные вкладки
- ✅ Сообщения об ошибках

### 5. 📱 Веб-версия
- **Адаптированная под веб** версия
- **Информация о водителях** без карты
- **Real-time обновления** через WebSocket
- **Пояснение** что карты доступны в мобильном приложении

## 🚀 Навигация

```
Первый запуск:
Language → Login → StudentTabs/DriverTabs

Повторный запуск (залогинен):
→ StudentTabs/DriverTabs (сразу)

Повторный запуск (не залогинен):
→ Login (язык уже выбран)
```

## 🎯 Функционал автовхода

1. **Первый вход**: Пользователь вводит ID → сохраняется токен
2. **Закрытие приложения**: Данные остаются в AsyncStorage
3. **Повторное открытие**: Автоматический переход на главный экран
4. **Выход**: Только данные авторизации удаляются, язык остается
