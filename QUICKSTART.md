## ❌ ЕСЛИ НЕ РАБОТАЕТ:

### Проблема: Приложение зависает при входе

**Решение 1:** Проверьте консоль в Expo Go
- Посмотрите сообщения об ошибках
- Если "Network Error" - backend недоступен

**Решение 2:** Убедитесь что телефон и компьютер в одной WiFi
- Компьютер IP: `192.168.151.235`
- Телефон должен быть в той же сети (192.168.151.x)

**Решение 3:** Проверьте что backend работает
```powershell
docker-compose ps
```
Все контейнеры должны быть UP

**Решение 4:** Перезапустите Expo сервер
```powershell
# Нажмите Ctrl+C в терминале bus
cd bus
npm start
```

### Проблема: "Неверный ID"

**Решение:** Создайте нового пользователя через админ-панель
1. Откройте http://localhost:3001
2. Войдите как админ (см. выше)
3. Вкладка "Студенты" → "Добавить студента"
4. Введите любой ID (например: 12345678)
5. Используйте этот ID в приложении
