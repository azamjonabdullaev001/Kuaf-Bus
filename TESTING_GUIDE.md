# 🧪 Быстрый Тест-Гайд

## Перед Тестированием

### 1. Убедитесь что бэкенд запущен:
```bash
# В корневой папке проекта
.\start-all.bat
```

Или вручную:
```bash
# Запустить backend
cd backend
go run main.go

# В другом терминале - запустить ngrok
ngrok http 8080
```

### 2. Обновите ngrok URL в config.js если нужно:
```bash
cd bus
# Откройте config.js и обновите URL из ngrok
```

## 🧪 Тестирование APK

### Шаг 1: Соберите новый APK
```bash
cd bus
eas build --platform android --profile preview
```

### Шаг 2: Установите APK на устройство

### Шаг 3: Тестирование Входа

#### ✅ Тест 1: Вход Студента
1. Откройте приложение
2. Введите университетский ID студента (например: `STU001`)
3. Нажмите "Войти"
4. **Ожидается:**
   - Запрос разрешения геолокации
   - Переход на карту студента
   - Фиолетовый маркер вашего местоположения
   - 3D автобусы водителей (если они онлайн)
   - НЕТ мигания карты

#### ✅ Тест 2: Вход Водителя
1. Откройте приложение
2. Введите университетский ID водителя (например: `DRV001`)
3. Нажмите "Войти"
4. **Ожидается:**
   - Запрос разрешения геолокации
   - Переход на карту водителя
   - 3D автобус вашего местоположения (золотой цвет)
   - 3D автобусы других водителей (если они онлайн)
   - Отображение скорости движения
   - НЕТ мигания карты

#### ❌ Тест 3: Неверный ID
1. Введите несуществующий ID (например: `WRONG123`)
2. Нажмите "Войти"
3. **Ожидается:**
   - Сообщение об ошибке: "Пользователь с ID "WRONG123" не найден"

#### ❌ Тест 4: Без Интернета
1. Отключите Wi-Fi/мобильный интернет
2. Попробуйте войти
3. **Ожидается:**
   - Сообщение: "Ошибка сети. Проверьте интернет-соединение"

## 📱 Тестирование Карты

### Тест Студента:

#### ✅ Тест 5: Отображение Себя
1. Войдите как студент
2. Дайте разрешение на геолокацию
3. **Ожидается:**
   - Фиолетовый маркер на вашей позиции
   - Карта центрируется на вас
   - Внизу координаты: `📍 Lat: XX.XXXXXX, Lon: XX.XXXXXX`
   - Вверху: `🚌 Активные водители: X`

#### ✅ Тест 6: Отображение Водителей
1. Убедитесь что хотя бы 1 водитель онлайн (войдите на другом устройстве)
2. **Ожидается:**
   - 3D автобусы на позициях водителей
   - Плавное движение автобусов при перемещении
   - НЕТ прыжков или мигания
   - Количество водителей обновляется

#### ✅ Тест 7: Движение
1. Начните двигаться (идите/едьте)
2. **Ожидается:**
   - Ваш маркер плавно движется
   - Карта НЕ центрируется (только при первой загрузке)
   - Координаты обновляются
   - НЕТ зависаний

### Тест Водителя:

#### ✅ Тест 8: Отображение Себя
1. Войдите как водитель
2. **Ожидается:**
   - 3D автобус (золотой) на вашей позиции
   - Координаты + скорость: `📍 Lat: XX, Lon: XX • XX км/ч`
   - Статус: `✓ Вы в сети`

#### ✅ Тест 9: Отображение Других Водителей
1. Войдите вторым водителем на другом устройстве
2. **Ожидается:**
   - На первом устройстве виден второй водитель
   - На втором устройстве виден первый водитель
   - Оба - 3D автобусы
   - Плавное движение
   - Количество обновляется

#### ✅ Тест 10: Скорость
1. Начните двигаться быстрее (на машине/велосипеде)
2. **Ожидается:**
   - Скорость обновляется в реальном времени
   - Формат: `• XX км/ч`
   - Маркер плавно движется

## 🐛 Возможные Проблемы и Решения

### Проблема: "Не могу войти"
**Решение:**
1. Проверьте логи - что именно пишет
2. Убедитесь что backend запущен
3. Проверьте ngrok URL в config.js
4. Убедитесь что пользователь существует в БД

### Проблема: "Карта не показывает моё местоположение"
**Решение:**
1. Проверьте что дали разрешение на геолокацию
2. Включите GPS на устройстве
3. Выйдите на улицу (GPS лучше работает снаружи)
4. Перезапустите приложение

### Проблема: "Не вижу водителей"
**Решение:**
1. Убедитесь что водители онлайн (вошли в приложение)
2. Проверьте WebSocket статус (вверху справа)
3. Должен быть зелёный: `● Подключено`
4. Если красный - проблема с сетью/backend

### Проблема: "Карта мигает"
**Решение:**
- Это НЕ должно происходить с новым кодом
- Если мигает - убедитесь что используете `GoogleStyleMap` а не `OpenStreetMap`
- Проверьте что файлы правильно обновлены

### Проблема: "Маркеры прыгают"
**Решение:**
- Это НЕ должно происходить
- Убедитесь что GPS точность хорошая (выйдите на улицу)
- Перезапустите приложение
